# Pharmacogenomics

A comprehensive pharmacogenomics analysis project for studying drug-gene interactions and personalized medicine applications.

## Project Structure

- **data/**: Input data files
- **scripts/**: Analysis and processing scripts
- **examples/**: Example data and usage examples
- **test/**: Test data and validation scripts
- **pharmgkb/**: PharmGKB database resources
- **source-allele-*** : Source data for allele definitions, frequencies, and functionality

## Key Files

- `star_alleles.json` - Star allele definitions
- `full-drug-database.xml` - Drug database
- `realistic_data_report.json` - Data quality reports

## Getting Started

1. Clone the repository
2. Review the documentation in `/docs` (if available)
3. Check example scripts in `/examples`

## Requirements

- Python 3.x
- Required packages listed in `requirements.txt` (if available)
